# 🔐 Random Password Generator

A simple and interactive **Random Password Generator** built using **HTML, CSS, and JavaScript**.
This project allows users to generate strong passwords instantly and copy them with one click.

---

## 🌙 UI Theme

This project features a **Dark Mode UI with a Purple Theme**, giving it a modern and clean look.

---

## 🚀 Features

* 🔑 Generate random secure passwords
* 📋 Copy password to clipboard
* ⚡ Instant password generation
* 🔔 Alert notification after copying
* 🎨 Clean and responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
📁 Random-Password-Generator
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How to Use

1. Clone the repository:

```
git clone https://github.com/afzaislamkz/Random-Password-Generator.git
```

2. Open the project folder

3. Run `index.html` in your browser

---

## 🎯 How It Works

* Click the **Generate** button
* A random password will appear in the input field
* Click the **copy icon** to copy the password
* A confirmation alert will be shown
